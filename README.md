# DZIOVYKLE

**Arduino NANO**
**Rotary encoder**
**LCD 128x68su ST7920 valdikliu**
**4 reliu modulis**

**750W sildytuvas**
**Ventiliatorius 12cm skersmens, nuputi ora nuo sildytuvo, ir judinti ora pacioje dziovykleje**
**Apsvietimas, 16W LED**
