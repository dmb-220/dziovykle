# DZIOVYKLE
dziovykle darzovems, vaisiams, ir grybams

Naudojama:
-**Arduino NANO**
-**Rotary encoder**
-**LCD 128x68su ST7920 valdikliu**
-**4 reliu modulis**
-**SHT30 temperaturos ir dregmes daviklis**
-**750W sildytuvas**
-**Ventiliatorius 12cm skersmens, nuputi ora nuo sildytuvo, ir judinti ora pacioje dziovykleje**
-**Apsvietimas, 16W LED**
